# Homebrew Tap for jig-sync

## Install

```bash
brew tap peanut-butter-tucker/tap
brew install jig-sync
```

## Usage

```bash
jig-sync init      # authenticate and set up
jig-sync push      # upload harness changes
jig-sync pull      # fetch profile updates
jig-sync status    # check sync state
jig-sync insights  # view analysis candidates
```

## Upgrade

```bash
brew upgrade jig-sync
```

## Uninstall

```bash
brew uninstall jig-sync
brew untap peanut-butter-tucker/tap
```
